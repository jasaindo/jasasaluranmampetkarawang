1. MAU AMANKAN DATA (BACKUP)?
   - Buka folder: C:\jasasaluranmampetkarawang
   - Klik kanan: selalu-ku-cadangkan.ps1
   - Pilih: "Run with PowerShell"
   - HASIL: File .zip baru muncul di C:\Backups-Saluran-Mampet

2. MAU BERSIHKAN / PULIHKAN BLOG (RESTORE)?
   - Buka folder: C:\Backups-Saluran-Mampet
   - Klik kanan: Pulihkan-Blog.ps1
   - Pilih: "Run with PowerShell"
   - HASIL: Folder proyek Anda yang rusak akan diganti yang baru.

3. MAU KIRIM KE INTERNET (SYNC)?
   - Buka PowerShell di folder proyek.
   - Ketik: git add .
   - Ketik: git commit -m "update"
   - Ketik: git push origin main
   - HASIL: Blog terupdate di Cloudflare.

CATATAN: 
Selalu pastikan ada file pnpm-lock.yaml di dalam folder proyek 
agar blog tidak error saat build.
